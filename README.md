# Ruby Server Side

- `brew install rbenv`
- `rbenv init`
- copy `eval "$(rbenv init -)"` and insert it in your shell file that rbenv init displays
- `rbenv install 2.6.3`
- `rbenv local 2.6.3`
- `gem install sinatra`

- `gem install sinatra-contrib` - automatically reloads the code when changes are made
